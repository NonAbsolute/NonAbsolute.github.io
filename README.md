## 开始项目

**首先请确保您安装了Node.js，并且拥有npm命令或者同等效用的其它命令**

本文档由Vuepress生成，Vercel提供云端部署服务



### 部署本地环境

在项目文件夹不存在node_modules目录时您需要运行：

```
npm install
```

### 本地环境调试

```
npm run docs:dev
```

### 本地环境编译
```
npm run docs:build
```
